# BombAway

## Description

BombAway, ursprünglich bekannt als Bomberman ist ein Multiplayer Spiel, basierend auf dem Konzept, den Gegenspieler zu vernichten. 
Unser Spiel basiert auf dem Framework Flask von Python. Über eine Website (URL) kann das Spiel gedownloadet werden. Der Server wird gestartet mit:
```python main.py```
Danach findet man das Spiel auf: ```localhost:2020```

Browserkompatibilität: ...
Programmiersprachen: Python 3, HTML 5, CSS, JavaScript
Ersteller: Matteo Miggano / Luka Joksimovic / David Abderhalden

## Codex

*Artikel 1*

Der Main Branch darf nur mit Erlaubnis aller Teammitglieder bearbeitet werden. (In Ausnahmefällen, bei Krankheit eines Teammitglieds, Unfall usw.)

*Artikel 2*

Alle Teammitglieder zeigen grossen Engagement und Interesse auf eine gute Arbeit. 

*Artikel 3*

Wenn jemand nicht weiter kommt oder Fragen hat, zögert er nicht auf die anderen zuzukommen und um Hilfe zu bitten.

*Artikel 4*

Wenn ein Teammitglied bereits fertig mit seiner Arbeit ist stellt er sich hilfsbereit gegenüber den Anderen. 

*Artikel 5*

Hat ein Teammitglied wegen geschäftlichen, schulischen oder privaten Gründen keine Zeit, wird das akzeptiert und die Arbeit verschoben (übernommen). 

*Artikel 6*

Es werden keine Viren, Würmer oder andere Maleware in unser Spiel implementiert :)

*Artikel 7*

Ein Teammitglied soll stehts verstehen, was es codiert. 

*Artikel 8*

Ein Script, das in unserer App implementiert ist, soll immer von mindestens einer Person des Teams verstanden werden. Ansonsten wird die App gelöscht und wir organisieren alle eine Übernachtungsparty als Projekt. 

*Artikel 9*

Mobbing ist erlaubt aber nur solange es keine Kosten auslöst. (Psychiater usw.)

*Artikel 10*

Arbeiten ist gut, aber über 4 Stunden wird ungesund!

*Artikel 11*

Niemand in unserem Team veröffentlicht das Spiel, versendet unseren Code oder missbraucht ihn für andere Zwecke bevor das Spiel fertiggestellt wurde. 

*Artikel 12*

In unserem Spiel sind weder sexistische, antisemitistische oder andere unerwünschte Anspielungen zu finden. 

*Artikel 13*

Wir sind gut gelaunt, auch wenn wir im TBZ sind. 

*Artikel 14*

Wenn unsere Noten unter einer 5 sind, schreibt jeder von uns einen Aufsatz zum Thema: Gute Noten und wesshalb man sie nicht bekommen hat. 

*Artikel 15*

Der Codex darf jeder Zeit angepasst, verändert und gelöscht werden. Wenn alle einverstanden sind damit! (Solange dieser Artikel noch nicht entfernt wurde)

*Artikel 16*

Die Person, die diesen Codex geschrieben hat, muss sämtliche Rechtliche konsequenzen selbst tragen.
